# BlazeDemo Performance Test

Projeto de testes de performance utilizando JMeter para validação do fluxo de compra de passagem aérea no site BlazeDemo.

## Objetivo

Validar o comportamento da aplicação sob carga e pico, garantindo:

- 250 requisições por segundo
- Tempo de resposta (P90) inferior a 2 segundos

## Stack

- JMeter
- Java (execução CLI)
